## 4. Chosen Export Strategy

### 4.1 CSV Export

**Strategy**: Native implementation with safe escaping

**Features**:
- Comma delimiter
- Quote escaping for fields containing commas/quotes
- UTF-8 BOM for Excel compatibility
- Native `Blob` and `URL.createObjectURL` for download

**No package required**.

---

### 4.2 Excel Export

**Strategy**: Use `xlsx` package (SheetJS)

**Features**:
- Generate `.xlsx` files
- Worksheet headers
- Export metadata (generated by, date, filters)
- Auto-width columns
- Professional styling

**Package**: `npm install xlsx`

---

### 4.3 PDF Export

**Strategy**: Use `jspdf` + `jspdf-autotable`

**Features**:
- A4 page format
- Enterprise report template
- Header with title, subtitle, generated by, date
- Filters summary (if applicable)
- Table data with auto-layout
- Page numbers
- Simple footer

**Package**: `npm install jspdf jspdf-autotable @types/jspdf-autotable`

**Note**: Basic template now. Letterhead/company branding will be upgraded in Phase 002F.

---

### 4.4 Print Export

**Strategy**: Browser print with clean printable view

**Implementation**:
- Generate a clean HTML document in a new window
- Include print-specific CSS
- Auto-trigger `window.print()`
- Close window after print dialog

**No package required**.

## 8. Security Impact Assessment

### 8.1 Security Requirements

✅ **No RLS Bypass**: Export utilities will only use data already loaded from RLS-protected server queries

✅ **No Service-Role Usage**: All export generation happens client-side after data is loaded

✅ **No Secret Exposure**: No `.env.local`, service keys, or auth metadata will be exported

✅ **Field Filtering**: Sensitive fields (`auth_user_id`, internal UUIDs, passwords) will be excluded from exports

✅ **Permission Checks**: Export menus will only appear if user has `.view` permission for that module

## 11. Risks & Mitigation

### Risk 1: MenuGroupContext Error (High)

**Risk**: Previous implementation had `MenuGroupContext` runtime error with `DropdownMenuLabel`.

**Mitigation**: 
- Use `DropdownMenuGroup` wrapper correctly
- Test dropdown in browser before deploying
- Follow patterns from fixed `organizations-table.tsx` and `branches-table.tsx`

### Risk 2: Large File Generation (Medium)

**Risk**: Generating PDF/Excel for large datasets (1000+ rows) may freeze browser.

**Mitigation**:
- Show loading state during export
- Limit export to current page/filtered data
- Add warning for large exports (future enhancement)

### Risk 3: Browser Compatibility (Low)

**Risk**: `Blob` and `URL.createObjectURL` may have issues in older browsers.

**Mitigation**:
- Use standard Web APIs supported by all modern browsers
- Test in Chrome, Edge, Firefox, Safari

### Risk 4: TypeScript Errors (Medium)

**Risk**: `jspdf-autotable` TypeScript types may be outdated or incomplete.

**Mitigation**:
- Install `@types/jspdf-autotable`
- Use type assertions if needed (`as any`)
- Test build with `npm run build`
