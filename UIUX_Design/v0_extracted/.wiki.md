# Project Summary
The Alliance Gulf Transport ERP project is a comprehensive web application designed to optimize logistics and transportation management. It provides businesses in the transport sector with a modern interface to track key performance indicators (KPIs), manage user accounts, and perform various administrative tasks efficiently. This ERP solution enhances operational visibility and streamlines workflows, making it a valuable tool for improving transportation operations.

# Project Module Description
The project includes several functional modules:
- **Login Page**: User authentication with a split-screen layout for branding and a clean login form.
- **Dashboard**: Offers a visual overview of essential KPIs and recent activities.
- **Sidebar**: A collapsible navigation menu for easy access to application sections.
- **Admin Users Page**: Manages user accounts with a professional data table.
- **Organizations Page**: Handles organizational structures and details.
- **Branches Page**: Manages branch locations and operational details.
- **Roles & Permissions Page**: Configures role-based access controls.
- **Audit Logs Page**: Tracks system activities and security events.
- **Dark Mode Support**: Allows users to switch between light and dark themes.
- **UI Kit**: A compressed archive containing reusable components and layouts for the ERP application.

# File Description Inventory
- **app/frontend/**: Contains all frontend source files, including components, pages, and configuration files.
- **assets/images/**: Stores image assets used in the application.
- **erp_project/**: The main directory for the project.
- **erp-ui-kit.tar.gz**: A compressed file containing reusable UI components and layouts for the ERP application.

# Technology Stack
- **React**: Frontend framework for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for faster development.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **ESLint**: Tool for identifying and fixing problems in JavaScript code.
- **@tanstack/react-query**: Library for managing server state in React applications.

# Usage
To get started with the project:
1. Install dependencies using the package manager.
2. Build the project to prepare it for deployment.
3. Run the application locally for development and testing.
4. To use the UI Kit, extract the compressed file and point your application to the extracted components.
