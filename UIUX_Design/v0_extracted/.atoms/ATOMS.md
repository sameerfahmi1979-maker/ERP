# Project Context

## Project Overview
UI/UX redesign of Alliance Gulf Transport ERP system. Converting from a basic Next.js/Supabase placeholder into a polished, modern enterprise application using React + Vite + shadcn/ui + Tailwind CSS.

## Key Decisions
| Date | Decision | By | Rationale |
|------|----------|-----|-----------|
| 2026-05-27 | Use blue-teal enterprise color palette | Alex | Professional, trustworthy feel for transport/logistics ERP |
| 2026-05-27 | Split-screen login design | Alex | Modern enterprise pattern with branding opportunity |
| 2026-05-27 | Dashboard-first approach with KPI cards | Alex | Immediate value visibility for users |

## Constraints
- Color Palette: Primary blue (#1e40af / blue-700), Accent teal (#0d9488 / teal-600), Success green (#16a34a), Warning amber (#d97706), Destructive red (#dc2626)
- Typography: Inter font family for UI, font-semibold for headings
- Dark mode support required
- Layout: Collapsible sidebar + header + main content area
- Module cards: Color-coded by category with status badges
- Images to Generate: company logo (transparent), login side panel gradient illustration, dashboard welcome illustration, empty state illustration
